# 🚀 Instructions to Push to GitHub

Follow these steps to upload your **TerraGuard Landslide Prediction Project** to GitHub.

### 1. Initialize Git in your project folder
Open a terminal in `c:\Users\WIN 10\Downloads\landslide` and run:
```bash
git init
```

### 2. Add all files
```bash
git add .
```

### 3. Create your first commit
```bash
git commit -m "Initial commit: TerraGuard Landslide Prediction Project"
```

### 4. Create a new repository on GitHub
- Go to [GitHub.com](https://github.com/new).
- Give it a name (e.g., `landslide-prediction`).
- Click **Create repository**.

### 5. Link your local folder to GitHub
*Replace `YOUR_USERNAME` and `YOUR_REPO_NAME` with your actual details:*
```bash
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
git branch -M main
```

### 6. Push your code
```bash
git push -u origin main
```

---
**Note:** If you get a login prompt, follow the GitHub instructions to authenticate using your browser or a Personal Access Token (PAT).
